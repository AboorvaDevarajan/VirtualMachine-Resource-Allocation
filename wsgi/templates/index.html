Help - VM allocator service

1. Refresh the page before generating the graph


2. Utilized PM resourses and needed VM resources (CPU, RAM and Network) % must be entered in the text box provided, as comma seperated values (it will have some values by default)


3. After entering the value click on generate graph button, once the graph is generated successfully "graph generated" message appears on screen


4. Now click on the RBF Graph, Bar Graph buttons to view the generated graph respectively. Make sure that the graph is generated before viewing the graph, Refresh the page for every update in the values
